基于wordpress+腾讯云服务器的网站搭建

写在开头，这个网站的诞生是由于笔者参加了学校的创业项目和“互联网+”创新大赛，由于笔者只学习了html+css的皮毛，因此这次尝试多抱有学习的目的，而这篇梳理也是一个新手的学习所得。

服务器和域名获取

这个项目笔者选用的是腾讯云的服务器和域名，阿里云和七牛云等平台也提供了相关服务，可以自由选择。在腾讯云的主页菜单中可以选择所需要的产品，就本项目而言，所需要的产品是一台云服务器和一个域名。

在此界面可以根据自己的需求选购云服务器，主要区别就是地域和机型。

在此界面可以注册自己想要的域名，但不能和其他域名产生重复。

其中比较重要的信息是主IPv4地址中带有“(公)”后缀的一串地址，即公钥，在之后的将域名和服务器绑定会用到它。

在这个界面可以管理自己的域名，腾讯云在注册的时候需要进行实名认证，同时域名注册之后也要进行实名认证才能进行其他操作。想让域名开始工作，需要手动为域名添加DNS解析，点击域名后面的解析即可添加。

其中最下面两条记录是域名自带的，需要手动添加两条记录，一条主机记录为@，另一条主机记录为www，记录值为刚才在云服务器中获取的公钥，添加完这两条记录，域名就会与云服务器绑定，在域名管理界面中现实域名服务状态为正常，域名就在正常工作。但在国内想使用域名，需要向工信部提交备案，备案分为个人性质和企业性质，所需材料也不一样，个人性质的主要材料为身份证，企业性质的主要材料为企业经营执照，本网站为个人性质，建议在备案审批成功后在添加域名解析，未备案网站如果运行有被查封的风险。腾讯云有网站备案的引导，总体过程比较简单，只需要提交相关材料，然后等待一段时间即可，不同地区的审批时间可能不尽相同，可能在3个工作日至15个工作日之间。同时在DNS解析添加界面可以下载网站的SSL证书，这在之后的搭建过程中需要用到，可以提早准备好。

至此，准备工作基本完成，购买了云服务器和域名，可以进行网站搭建了。

宝塔面板和wordpress部署

登录服务器之后安装宝塔面板

安装教程
1、登录腾讯云服务器，执行命令：ssh root@你的服务器IP；
2、执行安装命令：

yum install -y wget && wget -O install.sh http://download.bt.cn/install/install.sh && sh install.sh

3、执行安装命令后会提示：

Do you want to install Bt-Panel to the /www directory now?(y/n):

输入：y回车即可自动安装。
4、等待，大概2分钟左右，显示“Complete!”，即安装完毕！
5、安装完毕后会显示登录地址、用户名和密码：

登录地址为：http://你的服务器IP:8888 用户名：admin 密码：随机生成，安装完毕后会有显示

注意：请记录下你的用户名和密码，登录后台的时候会用到。
6、登录宝塔面板后台：http://你的服务器IP:8888
7、输入第5步中的账号和密码
8、登录后，会提示你安装Web环境，宝塔支持LNMP和LAMP环境，按需选择，点击一键安装.

在软件商店的一键部署中为网站部署wordpress。

在此界面为网站添加之前下载的SSL证书，否则网站在访问时会显示“不安全”。

之后注册wordpress账号，登录wordpress平台。建议收藏这个页面，方便对网站进行长期的维护。

之后的过程就请自行探索吧，所有diy的工作都会在这个平台进行，可以选择自己喜欢的模板，为网站添加自定义的内容，操作基本不涉及代码层面，几乎全是图形化的操作界面，对新手很友好，一个对前端代码一无所知的小白也能在很短的时间内搭建出属于自己的网站。值得一提的是，wordpress所附带的插件功能异常强大，在想实现一个功能却没有思路的时候，不妨去插件市场中寻找，通常都能找到想要的答案。

总结

整个网站的搭建运用模板建站的思想，模板建站工程周期短，效率高，已成为当今个人博客和中小企业企业站的首选解决方案，约有41%的个人站和企业站选用wordpress搭建，总体来说，这个项目的实现选择了较为主流且便利的方式，学习成本也较低。
